# Project Visuals References

These watercolor README visuals and the standalone donate QR card were prepared
for **Studio Noir - OmniVoice Playground**.

## Technology References

- **OmniVoice** by `k2-fsa` - the core speech synthesis and voice cloning model referenced by this playground.
- **PyTorch** by the PyTorch Foundation / Linux Foundation - tensor runtime used for CUDA/CPU inference.
- **FastAPI** by the FastAPI open-source project - Python backend API framework.
- **Uvicorn** by the Encode open-source ecosystem - ASGI server used to run the FastAPI app.
- **React** by Meta - frontend UI library.
- **Vite** by the Vite open-source project - frontend build and development tooling.
- **Tailwind CSS** by Tailwind Labs - utility-first styling system.
- **Lucide** by the Lucide open-source project - icon library used in the interface.
- **Hugging Face Hub** by Hugging Face - model weight distribution/cache ecosystem referenced by the backend loader.

## Generation Notes

- Watercolor backgrounds were generated with the built-in image generation workflow.
- The QR/payment card is kept separate from the watercolor visuals to preserve scan fidelity.
- Background prompts explicitly avoided generated text, logos, and synthetic QR codes.

## Prompt References

### Portrait

Watercolor and ink-wash portrait illustration of an AI voice synthesis tool: condenser microphone, layered waveform lanes, DAW timeline clips, voice parameter dials, and speech-to-audio motifs.

### Square

Watercolor square illustration of voice cloning and prompt-guided voice control: voice profile silhouettes, speech bubbles transforming into waveform streams, reference-audio upload motifs, neural connection lines, and control panels for pitch, style, age, and emotion.

### Banner

Landscape watercolor banner for an automated audio-video orchestration sandbox: wide DAW timeline, stacked audio clips, waveform grids, abstract video tracks, backend API flow lines, GPU acceleration glow, microphone, headphones, and model loading motifs.
